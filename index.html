HTML Tags

HTML tags are used to create elements in a web page. Tags are enclosed in angle brackets. For example:

<tag>Content</tag>

HTML Elements

HTML elements are the building blocks of web pages. An element typically consists of an opening tag, content, and a closing tag. Example:

<div>This is a div element.</div>

HTML Attributes

Attributes provide additional information about HTML elements. They are specified in the opening tag and have a name and value. Example:

<a href="https://www.example.com">Visit Example</a>

HTML Semantics

Semantic HTML elements clearly describe their meaning in a human- and machine-readable way. Examples:

<header>Header content</header>
<footer>Footer content</footer>

HTML Structure

A basic HTML document structure includes the following elements:

<!DOCTYPE html>
<html>
    <head></head>
    <body></body>
</html>

HTML Syntax

HTML syntax involves elements and attributes enclosed in angle brackets. Example:

<element attribute="value">Content</element>

Doctype Declaration

The doctype declaration defines the document type and version of HTML. For HTML5, it is:

<!DOCTYPE html>

Meta Tags

Meta tags provide metadata about the HTML document. Examples include charset, viewport, and description:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="A description of the page">
